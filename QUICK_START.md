# 🚀 Quick Start Guide

## For Next Time - Just Say:

**"Tile my pattern using the saved tool"**

Then upload your pattern PDF, and I'll automatically:
1. Use the tiling script from AI Drive
2. Process your pattern
3. Generate a tiled PDF with all alignment marks
4. Save it for you to download

## Example Commands:

### Simple:
> "I have a new pattern to tile, use the saved tool"

### With Options:
> "Tile this A0 pattern into Letter size pages using the saved tool"

> "Tile my pattern at 150 DPI for better quality"

## What I Need:

1. **Your pattern PDF** (any size - A0, A1, custom)
2. That's it! I'll handle the rest.

## Optional Settings:

You can customize:
- **Output size**: A4 (default) or Letter
- **DPI/Quality**: 100 (default), 75 (faster), 150 (better quality)
- **Output name**: Custom filename if you want

---

## File Locations:

- **Tool**: `/Pattern Tiling Tool/tile_pattern.py`
- **Instructions**: `/Pattern Tiling Tool/README.md`
- **Example Output**: `/Pattern Tiling Tool/Jacinta_Midi_Dress_Tiled.pdf`

💡 **Pro Tip**: The tool automatically calculates the grid size based on your pattern dimensions. No need to specify anything!
